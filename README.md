### 🧠 Czym się zajmuję

- **Lokalne modele LLM (offline)** — stawianie i zarządzanie modelami przez Ollama, integracja z VS Code (Continue.dev), praca w ograniczeniach sprzętu (GPU 4 GB VRAM).
- **RAG / wyszukiwanie semantyczne** — własny system offline (bge-m3 + indeks wektorowy) nad bazą wiedzy.
- **Automatyzacja i orkiestracja AI** — bramkowane pipeline'y badawcze, MCP, pamięć między sesjami, własne narzędzia CLI (scrapery + auto-notatki).
- **Dyscyplina weryfikacji** — najpierw szukam błędu w danych (lookahead / leakage), dopiero potem ogłaszam wynik.

> Wykształcenie techniczne (technik mechatronik) daje mi inżynierską dyscyplinę:
> **weryfikuję wyniki, zamiast im ufać.**
